# Axis Learning for Orientated Objects Detection in Aerial Images

由于航空图像中的鸟瞰图以及目标的各种尺度和任意角度，航空图像中定向目标检测仍然是一项具有挑战性的任务。大多数当前用于定向对象检测的方法是基于锚的，这需要大量预定义的锚并且耗时。在本文中，我们提出了一种新的单阶段无锚方法，以计算复杂度较低的逐像素预测方式检测定向对象。通过预测对象的轴来检测任意定向对象，该轴是连接对象头部和尾部的线，并且对象的宽度与轴垂直。通过直接在特征映射的像素级预测对象，该方法避免了设置与锚相关的多个超参数，并且计算效率高。此外，提出了一种新的宽高比感知方向中心度方法，以更好地加权正像素点，从而引导网络从复杂背景中学习区分特征，这为大宽高比目标检测带来了改进。该方法在两个常见的航空图像数据集上进行了测试，与大多数面向一级的方法和许多基于两级锚的方法相比，该方法具有更好的性能，且过程更简单，计算复杂度更低
